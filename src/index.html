Get Youtube Subcribers

Greeting: "Hello, this a backend API to get Youtube Subscriber" Routes are the folowing:

  • Base URL : http://localhost:3000
  • Route : Subcribers : " It will show the response with an array of subscribers(an Object) in JSON format"
  • Route : Subcribers/names : " It will show the response with an array of subscribers(an Object with only two fields name and subscribedChannel) in JSON format"
  • Route: "/subscribers/:id" : "Response with a subscriber*(an object)* with given id Response with status code 400 and Error message({message: error.message}) if id does not match",
